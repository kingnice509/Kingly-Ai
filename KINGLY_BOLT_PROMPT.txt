BUILD A COMPLETE AI-POWERED MUSIC CREATION AND CREATOR ECONOMY PLATFORM CALLED "KINGLY AI"

OVERVIEW:
Kingly AI is NOT just an AI music generator. It's a complete ecosystem where musicians, producers, and creators can:
- Create music (lyrics, beats, songs) with AI assistance
- Stream and discover music
- Buy and sell beats/lyrics in marketplace
- Earn money from multiple revenue streams
- Manage wallet and direct bank withdrawals
- Build their creator community

KEY FEATURES IMPLEMENTATION DETAILS
==============================================================================

AI MUSIC GENERATION:
- Integrate with OpenAI API for lyrics generation
- Integrate with Replicate API (MusicGen model) for beat/music generation
- Real-time preview of generated content
- Save generated content to user's library
- Option to regenerate, edit, or refine

PAYMENT PROCESSING:
- Integrate Paystack for payments (webhook for payment confirmation)
- Handle successful payments: Update wallet balance
- Process failed payments: Show error message
- Track all transactions in database

DIRECT BANK WITHDRAWAL:
1. User fills bank details form
2. System validates account using Paystack/bank API
3. Upon verification, system processes withdrawal
4. Update transaction status to "pending"
5. Once bank processes (24 hours), update to "completed"
6. Send confirmation email to user

USER AUTHENTICATION:
- Hash passwords with bcrypt
- Generate JWT tokens for session management
- Implement refresh token rotation
- Support OAuth (Google, Apple) using Passport.js
- OTP via SMS/email for phone login
- 2FA with authenticator app

MARKETPLACE COMMISSION:
- Platform takes 10-15% commission on sales
- Creator receives 85-90% of sale price
- Payment gateway takes 1.5-3% fee
- Track earnings in real-time

NOTIFICATIONS:
- Real-time notifications for purchases, follows, comments
- Email notifications for important events
- In-app notification bell with badge count
- Notification preferences in settings

SEARCH & DISCOVERY:
- Search by title, artist, genre, mood
- Trending algorithms (based on plays, likes, downloads)
- Recommendations based on listening history
- Browse by genre, mood, country
- Hashtag discovery
